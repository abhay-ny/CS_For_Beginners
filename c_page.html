Apna College

  • If you are a beginner and planning to learn C language, this is an excellent choice. In this YouTube channel, topics are taught not just by explaining the theory, but also by solving a lot of questions that will help you grasp the concepts concretely. And the friendly manner in which the course instructor, who is amicably known as Shradha did, makes you want to love this content.

Code With Harry

  • CodeWithHarry is a popular YouTube channel dedicated to teaching programming and web development concepts, primarily in Hindi. To learn C language, you can go for either the one-shot video, or the playlist. The things that would impress you would be the practice sheets, the cheat sheets and easy explanation of complex topics.

Neso Academy

  • Neso Academy's C programming language playlist is a well-structured resource for beginners looking to learn C from scratch. The playlist includes comprehensive topic coverage, structured learning and practical examples that makes learning more fun for beginners.

Great Learning

  • This website offers a variety of online courses on various trending topics. One such course is ‘C for Beginners’. It is an easy-to-understand course that teaches you the basics of C programming language. After completing the course, you are also eligible to earn a certificate.

w3school

  • This website is focused on teaching the users C language through small documentations and example codes on each topic, that you can run using W3Schools Tryit Editor. It contains great, detailed explanations and user-friendly UI. Also, users get to solve a many small exercises to test their grasp on the topics.
